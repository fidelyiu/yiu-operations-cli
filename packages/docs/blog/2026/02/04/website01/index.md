## 依赖

- [axum](https://crates.io/crates/axum) Axum 核心框架
- [axum_extra](https://crates.io/crates/axum-extra) Axum 的额外实用工具
  - typed-header 提供对 HTTP 头部的类型化处理
  - query 提供对查询参数的类型化处理
  - multipart 提供对表单数据的类型化处理
- [tokio](https://crates.io/crates/tokio) Async 运行时
  - [home](https://tokio.rs/)
  - [github](https://github.com/tokio-rs/tokio)
- [serde](https://crates.io/crates/serde) 序列化
  - [home](https://serde.rs/)
- [serde_json](https://crates.io/crates/serde_json)
- [tower](https://crates.io/crates/tower) Tower 生态系统
- [tower-http](https://crates.io/crates/tower-http) 用于 HTTP 客户端和服务器的 Tower 中间件和实用工具
  - cors 提供对 CORS（跨源资源共享）的支持
  - compression-gzip 提供对 HTTP 压缩（gzip）的支持
  - timeout 提供对请求超时的支持
  - trace 提供对请求和响应的日志记录功能
  - fs 提供对请求体和响应体的文件系统支持
  - limit 提供对请求体和响应体的限制功能
- [tower-service](https://crates.io/crates/tower-service) Tower 所基于的基础 Service 特征。
- [sqlx](https://crates.io/crates/sqlx) 数据库
- [jsonwebtoken](https://crates.io/crates/jsonwebtoken) 认证-JWT
- [argon2](https://crates.io/crates/argon2) 认证-Argon2 密码哈希算法
- [thiserror](https://crates.io/crates/thiserror) 错误处理
- [anyhow](https://crates.io/crates/anyhow) 错误处理
- [tracing](https://crates.io/crates/tracing) 日志 & 跟踪
- [tracing-subscriber](https://crates.io/crates/tracing-subscriber) 日志 & 跟踪
- [uuid](https://crates.io/crates/uuid) uuid
- [chrono](https://crates.io/crates/chrono) 支持时区的日期与时间处理
- [dotenvy](https://crates.io/crates/dotenvy) 对 dotenv crate 的良好维护的分支。
- [futures](https://crates.io/crates/futures) Rust 中的零开销异步编程
- [http-body-util](https://crates.io/crates/http-body-util) 对 HTTP 正文进行异步操作的 trait
